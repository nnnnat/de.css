# de.css
de.css is a default set of CSS Modules for designing mobile first interfaces
<hr>

## Usage

```npm i -S de.css``` or
```<link rel="stylesheet" href="https://unpkg.com/de.css/dist/de.min.css">```
## Features
* IE11+ support
* Mobile first
* Accessibility in mind
* 30ish KB
* No JS needed
